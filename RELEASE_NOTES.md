# 版本更新（Release Notes）

## 2026-07-18

- 新增 macOS 雙擊啟動器 `run_classify.command`，與 Windows 的 `run_classify.cmd` 對應——兩個平台都是雙擊即跑完整流程。
- 安裝說明（中英 README）拆成 Windows 與 macOS 兩版，SOP、疑難排解、流程圖同步更新。

## 2026-07-14

**存回應也能保留完整脈絡了。** 過去收藏的若是一則回應，筆記只有那一則的內容；現在：

- 筆記新增「上文脈絡」區：從母帖到你收藏那則回應的完整對話線，逐則標注作者。
- 筆記新增可摺疊的「回覆區」（Obsidian callout，預設收合）：原帖作者的回覆全數保留，其他留言自動過濾太短或純 emoji 的內容。
- 分類與標題生成會一併參考上文，存回應時的誤分類明顯減少。
- 筆記 frontmatter 新增 `saved_kind` 欄位，標記收藏的是原帖（`root`）還是回應（`reply`）。
- 新設定區塊 `thread-context`（`config.json`）：可整組關閉回到舊行為，或調整回覆的字數門檻與則數上限。

已知限制：回覆區只收匿名瀏覽可見的回覆，深層回覆與登入牆後的內容不會收錄。
